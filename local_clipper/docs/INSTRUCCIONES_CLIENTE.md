# CustosAI Clipper — Instrucciones de instalación

Guía para tus clientes. Puedes copiar esto en la descripción de Whop o incluirlo en el ZIP.

---

## macOS

1. **Descargar** `CustosAI-Clipper-macOS.zip` desde Whop
2. **Descomprimir** (doble clic en el ZIP)
3. **Abrir la app:**
   - Si macOS muestra *"LocalClipper.app no se puede abrir porque el desarrollador no está identificado"*:
     - Haz **clic derecho** en `LocalClipper.app`
     - Elige **Abrir**
     - En el diálogo, pulsa **Abrir** de nuevo
   - La primera vez basta con esto; después se abrirá con doble clic normal
4. (Opcional) Arrastra `LocalClipper.app` a la carpeta Aplicaciones

---

## Windows

1. **Descargar** `CustosAI-Clipper-Windows.zip` desde Whop
2. **Descomprimir** (clic derecho → Extraer todo)
3. **Ejecutar** `LocalClipper.exe`
   - Si Windows SmartScreen muestra una advertencia: **Más información** → **Ejecutar de todas formas**

---

## Requisitos

- **macOS**: 10.15 o superior (Catalina+)
- **Windows**: 10 o superior
- **Espacio**: ~2 GB libres (modelos de Whisper se descargan la primera vez)
- **Internet**: Necesario la primera vez para descargar el modelo de transcripción
